# Electrical Supply Store ⚡

![Status](https://img.shields.io/badge/Status-Em%20Desenvolvimento-yellow)
![GitHub language count](https://img.shields.io/github/languages/count/DenielCordeiro/Eletrical-supply-store)

Este projeto é o desenvolvimento de uma interface de e-commerce focada no setor de suprimentos elétricos. O objetivo é criar uma experiência de usuário limpa e funcional, permitindo a visualização de produtos e categorias de forma intuitiva.

## 🔗 Demonstração Online
Visualize o projeto em tempo real:  
👉 [CLIQUE AQUI PARA VER O PROJETO](https://denielcordeiro.github.io/Eletrical-supply-store/)
## 🚀 Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes ferramentas:

* **HTML5**: Estruturação semântica de toda a loja.
* **CSS3**: Estilização personalizada, focada em um layout moderno e profissional.
* **FontAwesome**: Ícones utilizados para melhorar a interface visual.

## 🛠️ Como executar o projeto

Você pode visualizar o projeto localmente seguindo um dos métodos abaixo:

### Opção 1: Download Direto (Mais simples)
1. Clique no botão verde **Code** no topo desta página e selecione **Download ZIP**.
2. Extraia o conteúdo do arquivo `.zip` em uma pasta de sua preferência.
3. Abra o arquivo `index.html` no seu navegador de preferência.

### Opção 2: Via VS Code (Recomendado para Devs)
1. Abra a pasta do projeto no seu **VS Code**.
2. Se tiver a extensão **Live Server** instalada, clique no botão "Go Live" no canto inferior direito ou use o atalho `Alt + L` + `Alt + O`.
